This project contains the Croatian localization providers for the Telerik Q1 2014 version of the following controls:

1.  Calendar
2.  ColorDialog
3.  CommandBar
4.  RadDock
5.  GanttView
6.  PivotGrid
7.  PrintDialogs
8.  RadGrid
9.  RadMessage
10. RadPageView
11. RadScheduler
12. SchedulerNavigator
13. RadSpellChecker
14. RadTimePicker
15. TreeView
16. RadWizard
