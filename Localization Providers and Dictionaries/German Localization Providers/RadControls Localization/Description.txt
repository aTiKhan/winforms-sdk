## Description

This project contains the german localization providers for the RadBrowseEditor, the RadColorDialog, the RadCommandBar, the RadDock, the RadGridView, the RadMarkupEditor, the RadMessageBox, the RadPageView, the RadPrintDialogs, the RadPropertyGrid, the RadReminder, the RadScheduler, the RadSchedulerNavigator, the RadSpellChecker, the RadTextBox, the RadTimePicker, the RadTreeView and the RadWizard for the Telerik Q3'12 Service Pack 1 - Release. The project also shows how to localize the RadRibbonBar, the RadCalenadar and the RadDateTimePicker. 

The demo project shows all of the previously named controls in action with the german localization providers.
